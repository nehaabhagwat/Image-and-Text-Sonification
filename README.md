# Image-and-Text-Sonification
sonifyTemperatures.py and sonifyGrayscaleImage.py need to be executed using JEM

1.	Download JythonMusic toolkit: https://jythonmusic.me/download/
2.	Read Jython documentation for creating Notes, Phrases and Melodies: https://jythonmusic.me/transcription/
3.  Average monthly temperatures for some location in the country over a long period can be retrieved from: https://www.ncdc.noaa.gov/
4.  This is an attempt to find ways to make apparent the trends of global warming through data sonification and visualization. 
5.  The second program sonifies grayscale images.
